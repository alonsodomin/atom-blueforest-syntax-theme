# Blueforest syntax theme

A dark blue theme inspired by the [Blueforest IntelliJ](https://github.com/sirthias/BlueForest) theme.

Supports following languages/markups:
 - Java
 - Scala
 - Python
 - Groovy
 - Ruby
 - JavaScript
 - HTML
 - XML
